# Question Bank — 質問項目 30 + 事業語言い換え

中小企業の経営者・マーケター向け。マーケ用語を避け、事業語で聞く。

## A. 事業の現在地（3問）

1. このサービスを一言で言うと？（社内の人に説明するように）
2. 今、一番売りたい商品/プランは？
3. 直近3か月で一番うまく売れたお客さんは、どんな人/会社でしたか？

## B. JTBD：決定打を掘る（5問、Bob Moesta 式）

4. 最近契約してくれたお客さんは、契約前は何で困っていましたか？
5. お客さんが「これだ」と決めた瞬間、何が決め手でしたか？
6. もしウチがなかったら、お客さんは何で代用しますか？
7. お客さんが検討した他社さんはどこでしたか？
8. 検討中に一番不安そうだったポイントは？

**深掘り**: 答えに「便利」「効率」など抽象語が出たら「具体的にどんな場面で？」「最後に同じことが起きたのはいつ？」と過去具体行動に戻す。

## C. ペイン / ゲイン（4問、VPC Customer Profile）

9. お客さんがよく口にする愚痴・困りごとは？
10. 契約後に「これは助かった」と言われたことは？
11. 他社さんと比べてよく褒められる点／negative に言われる点は？
12. お客さんがウチを誰かに紹介する時、どう説明していますか？

## D. サイトの役割（4問）

13. このサイトに来た人に、最終的にしてほしい1つの行動は？
14. どこから流入してくる人が多いですか？（広告/紹介/検索/SNS）
15. 月にどれくらい問い合わせ（CV）があれば「成功」と言えますか？
16. 既存サイトを見たお客さんからよく聞かれる質問は？

## E. オファー（4問）

17. 価格はどう設定していますか？（プラン構造、レンジ）
18. 保証・返金・無料体験はありますか？
19. 特典・キャンペーンは現在ありますか？
20. 競合と比べて価格はどのポジションですか？（安い / 中位 / 高い）

## F. 制約・前提（4問）

21. 公開希望日と、絶対に外せない日付は？
22. 載せられない情報（NG表現、未公開数値）は？
23. 既存の素材（写真/動画/原稿/ブランドガイド）はありますか？
24. 表現NG・参考にしてほしくないサイトはありますか？

## G. 競合・参考（3問）

25. 「このサイト好き」「こうしたい」と思う参考サイトは？（理由も）
26. 直接の競合だと思う会社は？
27. 「絶対こうはしたくない」サイトは？

## H. ブランド・トーン（3問）

28. ブランドの「らしさ」を一言で言うと？
29. お客さんに与えたい印象は？（プロっぽい/親しみやすい/革新的 など）
30. NG な表現・トーンは？

## マーケ用語 → 事業語 言い換え辞書

| マーケ用語 | 事業語の言い換え |
|---|---|
| ターゲットペルソナ | 「一番来てほしいお客さん」を一人、名前を付けて教えてください |
| バリュープロポジション | お客さんが「これウチに必要」と感じる一番のポイントは？ |
| KPI | 「今月よかったね」と言える数字は何ですか？ |
| ペインポイント | お客さんから一番よく聞く愚痴は？ |
| 競合優位性 | 他社さんと比べて「ここは負けない」と思うのは？ |
| CTA | サイトを見た人にまず何をしてほしいですか？ |
| エンゲージメント | サイトを見てる時にお客さんが「これ気になる」と思うやつを増やしたい、で合ってますか？ |
| アウェアネス | 「ウチをまだ知らないお客さん」と「もう知ってるお客さん」、どっちが多いですか？ |
| インサイト | お客さんが「言葉にしてないけど、たぶんこう思ってる」と感じることは？ |
| ICP（Ideal Customer Profile） | 一番ハマる「典型的な良いお客さん」の業種・規模・状況を1つだけ |
| ファネル | お客さんが「知る → 検討する → 買う」の3段階で、どこが一番詰まりますか？ |

## 質問運用ルール

- **一度に1問**。複数同時に聞かない
- 抽象的な答えが返ってきたら「具体的にいつ？」「直近の例で1つ」と過去具体に戻す
- マーケ語を口にしたら一拍置いて事業語に言い換える
- 答えにくそうな質問は順序を変える / 後回しにする
- 個人情報・機密が混入したら「これは載せても大丈夫ですか？」と確認
- カバレッジが偏ってきたら別ブロックに切り替える
